Пред- и постусловия

  • Определены на действиях
  • Предусловие
    • Должно быть выполнено до исполнения действия
  • Постусловие
    • Выполнено после исполнения действия

Оператор присваивания (1)

  • Определение
    • Обратная подстановка
    • // Pre: P[x→expr] (подставить expr вместо x)
x = expr;
// Post: P
  • Примеры
    • // Pre: b = 0
a = b;
// Post: a = 0

    • // Pre: a + a = b
a += a;
// Post: a = b
Георгий КорнеевПарадигмы программирования / Программирование по контракту12

Оператор присваивания (2)

  • Неверные подходы
    • Прямая подстановка
      • // Pre: P
x = expr;
// Post: P[x→expr]

      • // Pre: x = 1
x = 2;
// Post: 2 = 1
    • Простая конъюнкция
      • // Pre: P
x = expr;
// Post: P ∧ x = expr

      • // Pre: x = 5
x = x + 1;
// Post: x = 5 ∧ x = x + 1

Последовательность действий

  • из
    • // Pre: P1
S1
// Post: Q1

    • // Pre: P2
S2
// Post: Q2

    • Q1 ⇒ P2
  • следует
    • // Pre: P1
S1
S2
// Post: Q2
Георгий КорнеевПарадигмы программирования / Программирование по контракту14

Ветвление

  • из
    • // Pre: P ∧ cond
S1
// Post: Q

    • // Pre: P ∧ ¬cond
S2
// Post: Q
  • следует
    • // Pre: P
if (cond) S1 else S2
// Post: Q

Цикл

  • Из
    • // Pre: P ∧ cond
S
// Post: P
  • следует
    • // Pre: P
while (cond) S
// Post: P ∧ ¬cond
  • Цикл может не завершиться
    • Доказательство завершения

Стратегии доказательств

  • Сверху-вниз
    • От посылок к следствиям
    • Предусловие + действие ⇒ постусловие
  • Снизу-вверх
    • От следствий к необходимым посылкам
    • Постусловие + действие ⇒ необходимое предусловие

Инвариант цикла

  • int power(int a, int n) {
    int r = 1;
    // r' * a'^n' = a^n
    while (n != 0) {
        ...
    }
    // r' * a'^n' = a^n ∧ n' = 0
    // ⇒ r' = a^n
    return r;
}

Чистая функция

  • Результат зависит только от аргументов
  • Не имеет побочных эффектов
  • Пример
    • Функция double sqrt(double x)
Георгий КорнеевПарадигмы программирования / Программирование по контракту24

Предусловие

  • Условие, которое должно быть верно на момент вызова
    • Результат вызова с неверным предусловием не определен
  • Пример 
    // Pre: x > 0
double sqrt(double x)
Георгий КорнеевПарадигмы программирования / Программирование по контракту25

Постусловие

  • Условие, которое верно на момент возврата
    • Постусловие не выполнено ⇒ ошибка в программе
  • Пример 
    // Post: ℝ × ℝ = x ∧ ℝ ≥ 0
double sqrt(double x)

int value = 0;
 
// Pre: v ≥ 0
// Post: ℝ = value + v ∧ value' = value + v
int add(int v) {
    return value += v;
}

Сумма с аккумулированием (2)

  • Предусловие на неотрицательность value
    // Состояние
int value = 0;
 
// Pre: v ≥ 0 ∧ value ≥ 0
// Post: ℝ = value + v ∧ value' = value + v
int add(int v) {
    return value += v;
}

Сумма с аккумулированием (3)

  • Постусловие на неотрицательность value

int value = 0;
 
// Pre: v ≥ 0 ∧ value ≥ 0
// Post: ℝ = value + v ∧ value' = value + v
//      ∧ value ≥ 0

    return value += v;

Инвариант

  • Общая часть пред- и постусловия
    • Сохраняется всегда
  • Пример 
    // Inv: value ≥ 0

// Pre: v ≥ 0
// Post: ℝ = value + v ∧ value' = value + v

    return value += v;

Пример нескольких функций

  • int value;
 
void init() { value = 0; }

// Post: ℝ = value ∧ value' = value
