        bundle.getString("usage"),      // Usage:
        bundle.getString("options"),         // <options>
        bundle.getString("commands"),   // <commands>
        bundle.getString("Options"),        // Options:

    ));
}

Сравнение строк

  • Класс Collator
    • implements Comparator<String>
    • getInstance(locale?) – получение экземпляра
    • setStrength(int) – «сила» сравнения
      • PRIMARY – различать только буквы
      • SECONDARY – различать диакритику
      • TERTIARY – различать регистр
      • IDENTICAL – различать все символы
  • Класс RuleBasedCollator
    • Сравнивает строки в соответствии с заданными правилами
    • new RuleBasedCollator("c < e, E; é, É; ê = ě, Ê = Ě < F");
Georgiy KorneevТехнологии Java / Интернационализация и локализация41
