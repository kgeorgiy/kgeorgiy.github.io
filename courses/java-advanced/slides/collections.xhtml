Пример. Чтение в коллекцию

  • public int read(String file, Collection<String> c)
        throws IOException {
    Scanner scanner =
        new Scanner(Paths.get(file), "UTF-8");
    int read = 0;
    while (scanner.hasNext()) {
        read++;
        c.add(scanner.next());
    }
    return read;
}

Сравнение элементов

  • Метод object.equals(object)
  • Рефлексивность
    • x.equals(x)
  • Симметричность
    • x.equals(y) == y.equals(x)
  • Транзитивность
    • x.equals(y) ∧ y.equals(z) ⇒ x.equals(z)
  • Устойчивость
    • x.equals(y) не изменяется, если x и y не изменяются
  • Обработка null
    • x.equals(null) == false
Georgiy KorneevТехнологии Java / Collections Framework18

Пример equals (1)

  • Точка на плоскости

    protected int x;

        if (o instanceof Point) {
            Point that = (Point) o;
            return this.x == that.x && this.y == that.y;
        }
        return false;
    }
}
