Ошибки реализациия

  • Перепутан порядок implements и extends
  • Нельзя наследоваться от final класса
  • Метод интерфейса не может быть реализован protected методом
  • Метод setValue перегружен, но не реализован

  • Метод интерфейса не может быть реализован статическим методом
